# Task Manager API (Lab 2)

## Про проєкт
Це розширена версія REST API для системи управління завданнями.
У другій лабораторній роботі додано систему безпеки, ролі користувачів, завантаження файлів та автоматичну документацію.

##  Функціонал (Lab 2)
- **Аутентифікація:** Реєстрація та вхід через JWT (Access + Refresh Tokens).
- **Безпека:** Хешування паролів (bcrypt), захист заголовків (helmet), ліміт запитів (rate-limiter).
- **Файловий сервіс:** Завантаження аватарок/документів (multer) з валідацією типів.
- **Документація:** Інтерактивний Swagger UI.

##  Технології
- **Node.js & Express** - серверна частина.
- **PostgreSQL & Prisma ORM** - база даних.
- **JWT & Bcrypt** - безпека.
- **Multer** - обробка файлів.
- **Swagger** - документація API.
